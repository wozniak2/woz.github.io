Centralne Twierdzenie Graniczne

Mając daną wystarczająca dużą próbę z populacji ze skonczonym poziomem wariancji, rozkład danej zmiennej w tej probie upodabnia sie do rozkładu normalnego, a srednia z próby zmierza do średniej z całej populacji. Przy czym im wieksza próba tym odzwzorowanie lepsze.

A w praktyce?

  • nie musimy badać całej populacji, aby wiedzieć jak dana cecha się rozkłada w zbiorowości
  • średnie wartości danej cechy są najliczniej reprezentowane
  • im dalej oddalamy się od średniej, tym przypadków mniej

Rozkład normalny

najczęściej spotykany w przyrodzie rozkład zmiennych, np.:

  • testy szkolne,
  • testy inteligencji,
  • wzrost i waga populacji,
  • wynagrodzenia, wielkość produkcji, itp.

Oznaczanie rozkladu normalnego

\[ X \sim N(\mu, \sigma^2) ~~\mu - srednia, ~~ \sigma^2- wariancja \]

lub

\[ X \sim N(\mu, \sigma) \]

Im mniejsze odchylenie standardowe, tym rozklad jest bardziej skoncentrowany wokol sredniej oczekiwanej

# N(2,8)
# tworzymy sekwencje danych
x <- seq(-20, 20, by = 0.5)
# rysujemy rozkład normalny dla x
plot(x, dnorm(x,2,8), type="l", col="red", lwd=2, 
     ylim=c(0,0.4), main="funkcja gestosci",
     ylab="prawd.", xlab="x")
